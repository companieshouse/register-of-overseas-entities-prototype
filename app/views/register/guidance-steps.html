How to register an overseas entity

There are different steps to registering and this depends on who you are and who is registering for you. You may also be registering via our paper forms.

You are the agent and you are registering for an overseas entity online

You will be doing identity checks and registering via our online service

  1. Read the guidance and gather all data required
  2. Check if someone in your organisation has already requested an assurance code, if yes then skip to 6.
  3. Download the assurance code request form (one code per organisation)
  4. Email Companies House the completed assurance code request form
  5. Companies House emails you your organisation’s assurance code (do not share outside of your organisation)
  6. Complete the identity checks (see our guidance)
  7. Register the overseas entity and confirm identity checks information
  8. Once registered, you will receive an automatic response email with your application number
  9. Companies House emails you to confirm if the registration is successful

Who needs a code

UK-regulated agents that are completing the identity checks need a code. They must be based in the UK and supervised under the Money Laundering, Terrorist Financing and Transfer of Funds Regulations 2017. They can be individuals or corporate entities, such as:

  • credit institutions
  • financial institutions
  • auditors, insolvency practitioners, external accountants and tax advisers
  • independent legal professionals
  • trust or company service providers
  • estate agents and letting agents

How to get a code

You'll need to complete the 'Request an agent assurance code' document and email it to enquiries@companieshouse.gov.uk. Put ‘Agent assurance code’ in the subject line. This will help us to process your request more quickly.

Download 'Request an agent assurance code' (ODT, 9KB)

A Companies House examiner will check the document. If the request is accepted, they will email your code within a few working days. They may email you if there are any issues with the information you’ve provided.

How to use the code

If you’re carrying out identity checks and registering the overseas entity, you’ll need to enter the code as part of the Register an overseas entity and tell us about its beneficial owners service.

How to use the code if you are only carrying out identity checks

If someone else is registering the overseas entity, such as someone who works for the entity, you’ll need to complete and sign a statement that confirms the identity checks have been carried out. You’ll need to provide the code as part of this statement.

Download 'Overseas entity identity checks statement' (ODT, 13KB)

After the overseas entity has submitted its registration, you’ll have 14 days to email this statement to enquiries@companieshouse.gov.uk. Put 'Overseas entity ID checks statement' in the subject line. This will help us to process the registration more quickly.

We’ll check the statement and the application to register an overseas entity. If there are any issues with the information provided, we may email you.
