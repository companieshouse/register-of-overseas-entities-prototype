Guidance

Assurance codes for UK-regulated professionals

Overseas entities must have identity checks carried out on their beneficial owners and managing officers before they can be registered. This must be completed by a UK-regulated professional.

To confirm that this has been done, the UK-regulated professional needs an assurance code.

What an assurance code is

An assurance code is the equivalent of a signature. It confirms that the UK-regulated professional has carried out identity checks on an overseas entity.

Who needs a code

UK-regulated professionals that are completing the identity checks need a code. They must be based in the UK and supervised under the Money Laundering, Terrorist Financing and Transfer of Funds Regulations 2017. They can be an individual or a corporate entity, such as:

  • credit institutions
  • financial institutions
  • auditors, insolvency practitioners, external accountants and tax advisers
  • independent legal professionals
  • trust or company service providers
  • estate agents and letting agents

How to get a code

Use the Request an assurance code document to get a code. Then email it to enquiries@companieshouse.gov.uk with ‘Assurance code’ in the subject line. This will help us to process your request more quickly.

A Companies House examiner will check the document. If the request is accepted, they will email your code within a few working days. They may email you if any there are any issues with the information you’ve provided.

How to use the code

If you’re carrying out identity checks and registering the overseas entity, you’ll need to enter the code as part of the Register an overseas entity and tell us about its beneficial owners service.

How to use the code if you are only carrying out identity checks

If someone else is registering the overseas entity, such as someone who works for the entity, you’ll need to complete and sign a document that confirms the identity checks have been carried out. You’ll need to provide the code as part of this document.

After the overseas entity has submitted its registration, you’ll have X working days to email this document to enquiries@companieshouse.gov.uk. Include the 18 digit application reference number in the subject line. This will be in the confirmation email that is sent to the person who registers the overseas entity. This will help us to process the registration more quickly.

We’ll check the document and the application to register an overseas entity. If there are any issues with the information, we may email you.
