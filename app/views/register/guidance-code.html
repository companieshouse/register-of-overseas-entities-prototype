Guidance

Agent assurance codes for registering an overseas entity

A UK-regulated agent must carry out verification checks on beneficial owners and managing officers of an overseas entity before it can be registered. To confirm they are authorised to carry out the checks, the agent must request an agent assurance code.

What an agent assurance code is

An agent assurance code confirms that the UK-regulated agent has authorisation to carry out verification checks on an overseas entity. The agent needs to provide this code when the overseas entity is being registered.

If the UK-regulated agent is an organisation, you only need to request one code for the whole organisation. This code can then be used for every overseas entity that the agent carries out verification checks for. You should not share this code with anyone outside your organisation.

Who needs a code

UK-regulated agents that are completing the verification checks need a code. They must be based in the UK and supervised under the Money Laundering, Terrorist Financing and Transfer of Funds Regulations 2017. They can be individuals or corporate entities, such as:

  • credit institutions
  • financial institutions
  • auditors, insolvency practitioners, accountants and tax advisers
  • legal professionals
  • trust or company service providers
  • estate agents and letting agents

The agent that carries out the verification checks must not work at the overseas entity.

How to get a code

You'll need to complete the 'Request an agent assurance code' document and email it to agentcode@companieshouse.gov.uk. Put ‘Agent assurance code’ in the subject line. This will help us to process your request more quickly.

Download 'Request an agent assurance code' document (ODT, 9KB)

A Companies House examiner will check the document. If the request is accepted, we will email your code. We may email you if there are any issues with the information you’ve provided.

What to do if you lose your code or need a new one

Contact us to let us know and we’ll cancel your current code. Then you’ll need to request a new one.

How to use the code

If you’re carrying out verification checks and registering the overseas entity, you’ll need to enter the code as part of the Register an overseas entity and tell us about its beneficial owners service.

How to use the code if you are only carrying out verification checks

If someone else is registering the overseas entity, such as someone who works for the entity, you’ll need to complete and sign a statement that confirms the verification checks have been carried out. You’ll need to provide the code as part of this statement.

Download 'Overseas entity verification checks statement' document (ODT, 13KB)

After the overseas entity has submitted its registration, you’ll have 14 days to email this statement to agentcode@companieshouse.gov.uk. Put 'Overseas entity verification checks statement' in the subject line. This will help us to process the registration more quickly.

We’ll check the statement and the application to register an overseas entity. If there are any issues with the information provided, we may email you.
