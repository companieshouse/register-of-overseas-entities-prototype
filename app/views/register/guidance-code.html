Guidance

Agent assurance codes for registering an overseas entity

A UK-regulated agent must carry out identity checks on beneficial owners and managing officers of an overseas entity before it can be registered. To confirm this has been done, the agent must request an agent assurance code.

What an agent assurance code is

An agent assurance code is the equivalent of a signature. It confirms that the UK-regulated agent has carried out identity checks on an overseas entity.

The agent needs to provide this code when the overseas entity is being registered, even if the entity is registering itself.

Who needs a code

UK-regulated agents that are completing the identity checks need a code. They must be based in the UK and supervised under the Money Laundering, Terrorist Financing and Transfer of Funds Regulations 2017. They can be an individual or a corporate entity, such as:

  • credit institutions
  • financial institutions
  • auditors, insolvency practitioners, external accountants and tax advisers
  • independent legal professionals
  • trust or company service providers
  • estate agents and letting agents

How to get a code

Use the Request an agent assurance code document to get a code. Then email it to enquiries@companieshouse.gov.uk with ‘Agent assurance code’ in the subject line. This will help us to process your request more quickly.

A Companies House examiner will check the document. If the request is accepted, they will email your code within a few working days. They may email you if any there are any issues with the information you’ve provided.

How to use the code

If you’re carrying out identity checks and registering the overseas entity, you’ll need to enter the code as part of the Register an overseas entity and tell us about its beneficial owners service.

How to use the code if you are only carrying out identity checks

If someone else is registering the overseas entity, such as someone who works for the entity, you’ll need to complete and sign a statement that confirms the identity checks have been carried out. You’ll need to provide the code as part of this statement.

Download the Overseas entity ID checks statement

After the overseas entity has submitted its registration, you’ll have X working days to email this statement to enquiries@companieshouse.gov.uk. Put 'Overseas entity ID checks statement' in the subject line. This will help us to process the registration more quickly.

We’ll check the statement and the application to register an overseas entity. If there are any issues with the information provided, we may email you.
