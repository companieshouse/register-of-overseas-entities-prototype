Request an authorisation code for the registration of overseas entities

You need an authorisation code if you are a UK-regulated professional that is completing identity checks for overseas entities registering with Companies House.

To request a code, you'll need to provide:

  • contact information
  • the name of your supervisory body
  • your Anti-Money Laundering (AML) number, if applicable

Requesting a code takes around 5 minutes
